# Summit data

`summits.json` contains one record for each calendar year from 1975 through 2026.

The data distinguishes:

- factual metadata transcribed from source records;
- project-created summaries and policy-thread tags;
- links hosted by an issuing government (`tier: 1`);
- official summit text accessed through a documentary archive (`tier: 2`);
- known gaps that should not be filled by inference.

The annual collection is not a normalized inventory of ministerial or officials-level meetings. See `sources.html` for the complete scope and use note.
